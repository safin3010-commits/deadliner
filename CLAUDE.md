# Anti-Laziness Bot

## Запуск
- Python: `venv/bin/python3`
- Запуск: `bash start.sh`
- Логи: `tail -50 bot.log`

## Правила работы
- Перед изменением кода — сначала читать существующий файл
- Если текст для замены не найден — показать что реально есть, не гадать
- После применения изменений — проверять синтаксис через py_compile
- Писать результат: ✅ применено или ❌ не найдено
- Все изменения через patch-скрипты в /tmp/patch_*.py
- Никогда не создавать файлы через артефакты — только текст команд

## Структура
bot/handlers.py      — обработчики команд
bot/keyboards.py     — клавиатуры
bot/messages.py      — форматирование сообщений
scheduler.py         — все джобы и уведомления
grok.py              — клиент OpenRouter (DeepSeek V3/R1)
study_theory.py      — теория и английский
storage.py           — работа с задачами
reminders.py         — напоминания
parsers/lms.py       — парсер LMS ТюмГУ
parsers/modeus.py    — расписание Modeus
parsers/modeus_grades.py — оценки Modeus
parsers/messenger.py — Яндекс Мессенджер
parsers/mail.py      — Яндекс Почта
vk_browser.py        — парсер ВК через Playwright (БЕЗ токена)

## Ключевые детали
- Часовой пояс: UFA_TZ (UTC+5)
- Токен бота: TELEGRAM_TOKEN (не BOT_TOKEN)
- ИИ: ask_grok(prompt, system, smart=False) — smart=True для теории (R1)
- Тихие часы 00:00-09:00 — уведомления откладываются в pending
- Длинные сообщения: _send_long_message() режет по 4000 символов
- Символ ─ в f-строках ломает синтаксис — писать '─' * 20

## ВК парсер (vk_browser.py)
- Использует Playwright + Chrome + прокси 127.0.0.1:10808
- Куки хранятся в data/vk_cookies.json
- Обновление куки: venv/bin/python3 vk_browser.py --save-cookies
- Хэш по учебным ссылкам (mts-link.ru и др.) — стабильный
- Проверка один раз в день по дате (last_date в vk_seen.json)
- Если куки протухли — бот уведомит в Telegram

## Расписание уведомлений (UTC+5)
- 09:00 — утренний брифинг + дедлайны
- 09:30, 11:30, 13:30, 19:30, 21:30 — английский (#1-#5)
- 15:15 — теория по предмету (один раз в день)
- 21:00 — вечерний брифинг
- 23:00 — итоги дня
- вс 20:00 — недельный отчёт
- каждые 30 мин — ВК мониторинг (11:00-17:00)
- каждые 30 мин — LMS оценки
- каждый час — Modeus оценки
- каждые 10 мин — почта
- каждые 5 мин — мессенджер, напоминания, рандомная мотивация

## Что сделано (апрель 2026)
- ВК: Playwright вместо API токена, хэш по ссылкам, раз в день
- LMS: закрытие задач только по оценке
- Повторы: seen помечается ПОСЛЕ успешной отправки (mail, messenger, modeus)
- Стрики: полностью убраны из автоматических рассылок и handlers.py
- Категории: каждое уведомление начинается с жирного заголовка
- Факты дня: 35 IT-фактов, цикличный счётчик в data/facts_index.json
- IT-шутки: добавлены во все списки стилей мотивации
- Статистика: daily_stats.json — выполненные задачи по дням (90 дней)
- sent_notifications.json — дедупликация оценок и дедлайнов
- misfire_grace_time=7200 — пропущенные джобы запускаются в течение 2ч
- _clean_joke() — программная обрезка скобок и постскриптумов
- Английский: 40 правил грамматики цикличных, 500 слов история
