# ДедЛайнер 🤖

> Telegram-бот для студентов ТюмГУ + Нетология.  
> Следит за дедлайнами, расписанием и оценками — не даёт лениться.

![macOS](https://img.shields.io/badge/macOS-only-black?logo=apple)
![Python](https://img.shields.io/badge/Python-3.11+-blue?logo=python)
![Telegram](https://img.shields.io/badge/Telegram-Bot-blue?logo=telegram)

## Что умеет

- 🌅 Утренний / дневной / вечерний брифинг с расписанием и дедлайнами
- 📚 Синхронизация заданий из LMS и Нетологии
- 🎓 Оценки из Modeus и LMS — уведомление при появлении новой
- 📧 Уведомления о письмах (Яндекс Почта)
- 💬 Уведомления из Яндекс Мессенджера
- 📢 Мониторинг беседы ВКонтакте
- 🇬🇧 Английский язык — слово и правило 3 раза в день
- 🧠 Теория по предметам в учебные дни
- 🤖 AI-анализ учёбы и мотивация через Groq
- 📝 `/quiz` — парсинг вопросов тестов LMS во время прохождения

## Расписание уведомлений

| Время | Что приходит |
|-------|-------------|
| 9:00  | Утренний брифинг — погода, расписание, дедлайны, цитата |
| 9:30  | Английский #1 — слово + правило |
| 11:00 | Теория по предмету (только в учебные дни) |
| 13:00 | Цитата дня |
| 14:00 | Дневной брифинг — задача дня + мотивация от AI |
| 15:30 | Английский #2 |
| 17:00 | Напоминалка + 3 ближайших дедлайна |
| 21:00 | Вечерний брифинг — итоги дня, завтра, анализ учёбы |
| 21:30 | Английский #3 |
| Вс 20:00 | Недельный отчёт |

Фоновые проверки: почта каждые 5 мин, мессенджер каждые 5 мин, ВК каждые 15 мин, оценки каждые 10 мин.

Тихие часы: 00:00–09:00 — уведомления не отправляются.

## Быстрый старт

### 1. Требования

- macOS
- Python 3.11+
- Google Chrome (только для `/quiz`)

### 2. Клонируй репозиторий

```bash
git clone https://github.com/safin3010-commits/deadliner.git
cd deadliner
```

### 3. Запусти установщик

```bash
python3 setup.py
```

Установщик сам спросит все нужные данные, проверит логины и пароли, создаст `.env` и установит зависимости.

### 4. Запусти бота

```bash
bash start.sh
```

Бот работает в фоне. Терминал можно закрыть.

## Что спрашивает установщик

| Шаг | Что нужно | Где взять |
|-----|-----------|-----------|
| Имя | Как бот будет к тебе обращаться | Любое |
| Telegram токен | Токен бота | @BotFather → /newbot |
| Telegram ID | Твой ID | @userinfobot |
| Город | Для погоды и часового пояса | Вводишь название |
| Groq ключ | Для AI-функций | console.groq.com → API Keys |
| Логин utmn.ru | Для Modeus и LMS | Твои учётные данные ТюмГУ |
| Нетология | Email и пароль | Твои данные netology.ru |
| Яндекс Почта | Почта + пароль приложения | Яндекс ID → Безопасность → Пароли приложений |
| Яндекс Мессенджер | Войти в браузере | Откроется автоматически |
| ВКонтакте | Ссылка на беседу | Из адресной строки браузера |

Всё опциональное кроме Telegram токена, ID и Groq ключа.

## Проблемы

**Бот не запускается**
```bash
tail -50 bot.log
```

**Расписание не работает**  
Проверь `MODEUS_USERNAME` и `MODEUS_PASSWORD` в `.env`

**Оценки не приходят**  
Проверь `LMS_USERNAME/PASSWORD` и `MODEUS_USERNAME/PASSWORD` в `.env`

**AI не отвечает**  
Проверь `GROQ_KEY_1` на [console.groq.com](https://console.groq.com) — возможно превышен лимит

**Quiz не находит вопросы**  
Убедись что залогинен в LMS через Chrome (не через другой браузер)

**ВК не отправляет сообщения**  
Куки протухли — запусти: `venv/bin/python3 parsers/vk_browser.py --save-cookies`

**Яндекс Мессенджер не работает**  
Запусти установщик заново: `python3 setup.py` и перенастрой мессенджер

## Как пользоваться ботом

### Главное меню

После запуска бот покажет кнопки. Вот что они делают:

| Кнопка | Что делает |
|--------|-----------|
| 📋 Задачи | Список всех активных задач с дедлайнами |
| ✅ Выполнено | Отметить задачу как выполненную |
| ➕ Новая задача | Создать задачу вручную |
| 🔔 Напоминания | Список активных напоминаний |
| ➕ Напоминание | Создать напоминание |
| 📊 Оценки | Текущие оценки по предметам |
| 📅 Расписание | Расписание на сегодня и завтра |
| 🤖 Анализ | AI-анализ успеваемости |

### Как создать задачу

**Способ 1 — через кнопку:**
1. Нажми ➕ Новая задача
2. Введи название
3. Введи дедлайн в формате `27.04.2026` или `27.04.2026 23:59`

**Способ 2 — просто написать боту:**

Бот сам распознаёт задачи из обычного текста:

- `Сдать лабораторную до пятницы`
- `Добавь на завтра: матан, физра, английский`
- `Купить учебник до 30 апреля, напомни за 3 дня`
- `Каждый день на эту неделю: повторить конспекты`

Бот сам разберёт дату, создаст задачу и поставит напоминание если попросишь.

### Как создать напоминание

1. Нажми ➕ Напоминание
2. Введи текст напоминания
3. Введи дату и время в формате `27.04.2026 18:00`
4. В указанное время придёт уведомление

### Как запустить /quiz

1. Открой LMS в Google Chrome и войди в аккаунт
2. Напиши боту `/quiz`
3. Открой нужный тест в браузере — вопросы начнут приходить автоматически
4. После окончания теста напиши `/quizstop`

> ⚠️ Quiz работает только с Google Chrome. Другие браузеры не поддерживаются.
